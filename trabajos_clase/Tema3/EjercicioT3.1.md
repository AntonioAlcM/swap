#Ejercicio 3.1
Buscar con qué órdenes de terminal o herramientas gráficas podemos configurar bajo windows y bajo linux el enrutamiento del tráfico de un servidor para pasar el tráfico desde una subred a otra

##**Linux**
He encontrado el comando route, tiene opciones para añadir o eliminar rutas, con ip route también puedes establecer rutas.
También modificando el archivo /etc/network/interfaces puedes añadir una ruta o eliminarla.

##**Windows**
Con Enrutamiento y Acceso Remoto podemos pasar el tráfico de una red a otra.
En la consola usando el comando Arp puedes añadir y eliminar rutas.
